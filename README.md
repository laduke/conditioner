# Conditioner
View the important local conditions on one page. 

-  Sunrise & Sunset
-  Tides
-  Wind
-  Swell Info
-  Water Temperature
-  Air Temperature

# Technologies 
- ES6 
- Express
- React
- Redux
- Ramda
- Material-UI
- React-flexbox-grid
- Tape
- Webpack

# Developer setup
Written on node 6.4 and uses ES6 features in node and browser. 

- clone
- `npm install`
- `npm run dev`
- open http://localhost:8080/
